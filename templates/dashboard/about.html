About
OCTAD is aimed to build up an open community (open data, open code and open science) for cancer therapeutic discovery. It is hosted and maintained by Bin Chen Lab @ Michigan State University. The main contributors of version 1.0 are as follows.
Anita Wen (desktop version/use case)
Anthony Sciarini (front end/html page format)
Ben Glicksberg (front end/user experience)
Billy Zeng (back end)
Bin Chen (lead, PI)
Eugene Chekalin (R package)
Jing Xing (back end/pharmacology)
Ke Liu (data sets)
MSU IT (portal)
Optra Health Inc, iPhronesisTM (front end)
Patrick Newbury (use case, front end, back end)

Contact octad.contact@gmail.com for any questions.
